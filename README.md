
# Evolve AI

Evolve AI is a futuristic goal-tracking and self-improvement web app that helps users stay motivated by interacting with their future successful self.

## Features

Modern dark-themed UI Goal onboarding flow Dashboard with progress tracking Daily mission system Visual roadmap Future Me chat interface Achievement system Profile and progress tracking Local storage support ##Tech Stack HTML CSS JavaScript Local Storage ##Project Status This project is currently a learning project and active work in progress.

Some features are fully implemented while others are being improved and connected together.
